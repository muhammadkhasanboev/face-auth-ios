# Project Structure

## Repository Layout

*   **Root Directory**: Contains `project.yml`, the generated Xcode project, and workspace data.
*   **TadSigningDemo/**: Primary source folder with application logic, assets, and configuration files.
*   **Packages/**: Referenced logically to include the local `TadSigningSDK`.

## Configuration & Project Generation

The project uses **XcodeGen** to manage the `.xcodeproj` file.

### XcodeGen (project.yml)
*   **Target Type**: Application for iOS 16.0.
*   **Bundle Identifier**: `uz.tad.TadSigningDemo`.
*   **Entitlements**: `com.apple.developer.associated-domains` → `webcredentials:signing.tadi.uz`.
*   **Info.plist**: Injects `NSCameraUsageDescription` and `NSFaceIDUsageDescription`.

## Dependency Management: TadSigningSDK

The project relies on a local reference to the `TadSigningSDK` package (`XCLocalSwiftPackageReference`).

## Assets and Resources

*   **AppIcon**: Managed via `AppIcon.appiconset` (1024×1024 universal).
*   **AccentColor**: Defined in `AccentColor.colorset`.
*   **Launch Screen**: Configured via `UILaunchScreen` key in `Info.plist`.
