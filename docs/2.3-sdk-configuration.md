# SDK Configuration (SDKConfig.swift)

`SDKConfig.swift` is the central configuration hub for `TadSigningSDK`. It uses a static `enum` namespace pattern to provide a pre-configured `TadSigningConfig` instance.

## TadSigningConfig Parameters

| Parameter | Value | Description |
| :--- | :--- | :--- |
| `apiBaseUrl` | `https://signing.tadi.uz` | Base URL for backend API services. |
| `publicKeyPem` | ES512 PEM String | Public key for verifying server-side signatures (ES512/ECDSA P-521). |
| `rpId` | `signing.tadi.uz` | Relying Party Identifier for Passkey/WebAuthn operations. |
| `serviceName` | `tad-signing-demo` | Unique identifier for the client application. |
| `blockProxy` | `true` | Prevents traffic interception via MITM proxies (Charles, Burp Suite). |

## Security Notes

*   **ES512 Public Key**: Critical for verifying that data received from `apiBaseUrl` has not been tampered with.
*   **Proxy Blocking**: When `blockProxy: true`, the SDK detects and blocks HTTP proxies commonly used in reverse engineering attacks.
*   **`rpId`**: Must match the domain in the `associated-domains` entitlement and the AASA file on the server.
