# App Entry Point (App.swift)

The `App.swift` file is the foundational entry point for the **TadSigningDemo** application using the SwiftUI lifecycle.

## Implementation Details

The `TadSigningDemoApp` struct conforms to the `App` protocol and defines the `body` property returning a `Scene`.

*   **WindowGroup**: Represents the primary window of the application and manages the lifecycle of the window.
*   **Root View Injection**: `ContentView` is instantiated inside the `WindowGroup` as the root of the entire application's UI.

| SwiftUI Concept | Code Entity |
| :--- | :--- |
| **Entry Point** | `@main` |
| **App Definition** | `struct TadSigningDemoApp` |
| **Primary Scene** | `WindowGroup` |
| **Initial View** | `ContentView()` |

The simplicity of this entry point ensures that application launch is lightweight, deferring SDK configuration and UI rendering to subsequent layers.
